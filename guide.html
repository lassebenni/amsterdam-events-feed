1. Log into WordPress

Open your browser and go to http://localhost:8080/wp-admin.

2. Install and Activate the Feedzy Plugin

  1. Navigate to Plugins > Add New.
  2. Search for Feedzy RSS Feeds.
  1. Click Install Now on the Feedzy RSS Feeds card.
  1. After installation, click Activate.

3. Create the Events Page

  1. Go to Pages > Add New.
  1. Click Add New.

  1. Enter Events as the page title.

  2. In the content editor, paste the Feedzy shortcode:

    [feedzy-rss feeds="http://localhost:8080/wp-content/themes/custom/events.xml" max="12"]
  1. Click Publish.

4. Set as Static Front Page

  1. Navigate to Settings > Reading.
  2. Under Your homepage displays, select A static page.
  3. Choose Events for the Homepage.
  4. Click Save Changes.

5. View the Live Events Feed

Open http://localhost:8080 in your browser to see the events grid.

üîÑ Updating the Feed

When you need to refresh the events:

  1. Regenerate your events.xml feed (e.g., run make scrape).
  2. Ensure events.xml is served from /wp-content/themes/custom/events.xml.
  3. Reload your homepage to see the latest events.
